GPU Accelerated Data Science

GPU accelerated data science

What is RAPIDS

NVIDIA GPU based data science libraries in such as cuDF (similar to Pandas), cuML (similar to scikit-learn), and cuGraph (similar to networkX). Learn more about our 14+ libraries and more about hardware-software requirements.

Why Use RAPIDS

Spend less time training and moving data, more time optimizing and working. 20x - 100billion X performance increase over CPU with TCOs up to 7x. RAPIDS enables End to end GPU data science, with familiar APIs and libraries that seamlessly work together. Use RAPIDS with a local desktop, in the cloud, or scale out on large clusters. Learn more about RAPIDS acceleration.

Quick Start

Quick Local Install

RAPIDS offers several installation methods, the quickest is shown below.

Requirements

A. Compatible NVIDIA GPUs

B. Ubuntu 18.04/20.04, CentOS, Rocky Linux 8 or via WSL2 on Windows 11.

C. The latest CUDA and NVIDIA drivers. Check yours with: nvidia-smi

Install with Conda

1. Download and run the conda install script: 

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh

2. Then quick install rapids with:

conda create -n rapids-22.08 -c rapidsai -c nvidia -c conda-forge \ rapids=22.08 python=3.9 cudatoolkit=11.5

Install with PIP

1. Install via pip channels:

pip install cudf-cu11 dask-cudf-cu11 --extra-index-url=https://pypi.ngc.nvidia.com
pip install cuml-cu11 --extra-index-url=https://pypi.ngc.nvidia.com
pip install cugraph-cu11 --extra-index-url=https://pypi.ngc.nvidia.com

Do I need an Advanced Install?

If you want to use Docker, have an uncommon OS, hardware configuration, environment, or need specific versioning such as CUDA 11.5 or Python 3.7 then you should use the installation selector in our Advanced Installations Page
